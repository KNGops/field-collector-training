GitHub Pages Deploy Guide

1. Generate codes in the Code Generator tab.
2. Copy the hash snippet from the output.
3. Open kng_mms_training_v9.html in a text editor.
4. Find const VALID_CODE_HASHES = [...]
5. Replace the entire array with the new snippet.
6. Commit & push to your GitHub Pages repo.
7. Distribute codes to collectors via your roster CSV.

Data & Privacy

This admin dashboard is a 100% client-side, static HTML file. No data is ever sent to a server.

Auth layer 1 — URL hash: The secret in the bookmark URL is never sent to servers or indexed by search engines. Rotate it by updating REQUIRED_HASH in the source.
Auth layer 2 — PIN: Stored in localStorage on this device only. Change it above at any time.
